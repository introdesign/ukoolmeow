# Backend Setup

## Installation

```bash
npm install
```

## Configuration

1. Create a MongoDB Atlas account at https://www.mongodb.com/cloud/atlas
2. Create a new cluster
3. Get your connection string
4. Update `.env` file with your MongoDB URI

## Running the Server

Development mode:

```bash
npm run dev
```

Production mode:

```bash
npm start
```

Server will run on http://localhost:5000
